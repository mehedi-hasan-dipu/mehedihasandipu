StayMate – Hostel & Mess Finder (Java Desktop App)  
A modern, user-friendly desktop application built with **pure Java Swing** that helps students quickly find and book safe, affordable hostels and messes in **Dhaka** and **Rajshahi**, Bangladesh.

Features
- Beautiful UI with splash screen, login/registration (skippable)  
- Powerful search + advanced filters (price, room type, gender, location)  
- 29+ real, verified hostels (8 premium Dhaka + 21 authentic Rajshahi hostels)  
- Detailed hostel profiles with facilities, contact, address & one-click Google Maps  
- Simulated manager chat with persistent history  
- Instant booking confirmation  
- 100% offline – no internet required after download  

Tech Stack
- Java (JDK 17+)  
- Java Swing & AWT (pure, no external libraries)  
- In-memory database (ArrayList + HashMap)  

Perfect For
University students (RUET, RU, Varendra, Medical College, DU, NSU, BRAC, etc.) looking for trusted accommodation without brokers or fake Facebook posts.

**Ready to run
**
Run `Main.java` or the compiled JAR and start finding your perfect stay in seconds!

Feel free to contribute more hostels, features, or an Android version!  

Made with passion for students, by a student.
